# LinkedData.Center SDaaS community edition scenario

Try it https://en.linkeddata.center/l/sdaas-ce-demo/

This is a tutorial to teach you how to build a knowledge base using the community edition of Smart Data as a Service Platform by LinkedData.Center.

It shows how to integrate some Linked Open Data source to create a [KEES compliant knowledge bases](http://linkeddata.center/kees) that answers to a set of questions.
